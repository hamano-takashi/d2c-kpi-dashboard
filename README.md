# D2C KPI Management Dashboard

目標売上13億円達成のためのKGI/KPI管理ダッシュボードシステム

## 構成ファイル

```
d2c-kpi-dashboard/
├── index.html              # Webダッシュボード（メイン）
├── kpi-data.json          # KPIマスターデータ（261項目）
├── spreadsheet-template.csv # スプレッドシート用テンプレート
└── README.md              # このファイル
```

## クイックスタート

### 方法1: ローカルで確認

1. `index.html` をブラウザで開く
2. サマリー画面で全体状況を確認
3. 各タブでエージェント別詳細を確認
4. 「実績入力」タブで数値を入力

### 方法2: Google スプレッドシートにインポート

1. Google スプレッドシートを新規作成
2. 「ファイル」→「インポート」→「アップロード」
3. `spreadsheet-template.csv` をアップロード
4. 「現在のシートを置換」を選択してインポート

## 機能

### 1. サマリーダッシュボード
- KGI達成状況（年商/営業利益/LTV÷CPA/リピート率）
- エージェント別達成スコア
- 要注意項目アラート
- レーダーチャート

### 2. 実績入力
- 主要KGIの手動入力
- ローカルストレージに自動保存

### 3. エージェント別詳細
- 全261 KPIの目標値・実績値・達成率
- 現実性チェック（ベンチマーク比較）
- カテゴリ別表示

### 4. 現実性チェック機能

目標値が業界ベンチマークと比較して妥当かを自動判定

| 判定 | 基準 |
|------|------|
| ✅ 妥当 | ベンチマーク範囲内 |
| ⚠️ 要検討 | ベンチマーク下限未満 |
| 🔴 非現実的 | ベンチマーク上限超過 |

## カスタマイズ

### 目標値の変更

`kpi-data.json` の各KPIの `target` 値を編集

```json
{
  "id": "cmd_001",
  "name": "年商",
  "target": 1300000000,  // ← ここを変更
  "unit": "円",
  "benchmark_min": 1000000000,
  "benchmark_max": 1500000000
}
```

### ベンチマークの調整

業界や商材に合わせて `benchmark_min` / `benchmark_max` を調整

## クライアント提供時の手順

### スプレッドシートで提供

1. `spreadsheet-template.csv` をインポートした Google スプレッドシートを作成
2. 「共有」でクライアントに閲覧/編集権限を付与
3. クライアントは「実績値」列に数値を入力
4. 「達成率」列の数式で自動計算

### 達成率の数式（スプレッドシート用）

```
=IF(F2="",0,IF(E2<>0,H2/E2*100,0))
```
（E列=目標値、H列=実績値の場合）

### 現実性判定の数式（スプレッドシート用）

```
=IF(H2="","",IF(H2<J2,"⚠️ 下限未満",IF(H2>K2,"🔴 上限超過","✅ 妥当")))
```
（H列=実績値、J列=ベンチマーク下限、K列=ベンチマーク上限の場合）

## Webホスティング（オプション）

### GitHub Pages

1. GitHubリポジトリを作成
2. `index.html` と `kpi-data.json` をプッシュ
3. Settings → Pages → Source: main → Save
4. `https://[username].github.io/[repo-name]/` でアクセス可能

### Google Apps Script（GAS）

スプレッドシートと連携する場合は、GASでWebアプリ化も可能

# Changelog

## [v2.0.0] - 2025-01-20

### マルチテナント対応

#### 新機能
- **スーパー管理者機能**
  - スーパー管理者ダッシュボード（テナント一覧・統計）
  - 初回セットアップ機能
  - テナント作成・編集・削除
  - テナント一時停止・有効化

- **マルチテナント管理**
  - テナント別データ完全分離
  - テナント詳細ページ（ユーザー・プロジェクト・招待履歴）
  - ユーザー招待リンク生成
  - ユーザー削除機能
  - テナント完全削除機能

- **招待登録システム**
  - 招待リンクによるユーザー登録
  - 権限設定（管理者・編集者・閲覧者）
  - 有効期限管理

- **初期KPIテンプレート**
  - テナント作成時に自動適用
  - EC/D2C向け標準KPIツリー構造

#### データベース
- テナントテーブル追加
- 招待テーブル追加
- スーパー管理者テーブル追加
- SQLite/PostgreSQL両対応

#### UI/UX改善
- 既存CSSクラスによる統一デザイン
- レスポンシブ対応
- モダンなカード・テーブルレイアウト

- Render.com対応（render.yaml）
- Supabase PostgreSQL対応

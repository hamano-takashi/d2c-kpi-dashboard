# Changelog

## [v1.0.0] - 2024-01-19

### 初回リリース

#### 機能
- KPIツリー構造（売上 = 集客 × CVR × 顧客単価 × LTV）
- KPI追加・編集・削除機能
- 目標値・実績値入力
- ユーザー認証（JWT）
- プロジェクト管理
- メンバー招待・権限管理（admin/editor/viewer）

#### データベース
- SQLite永続化（ローカル環境）
- PostgreSQL対応（本番環境）

#### デプロイ
- Render.com対応（render.yaml）
- Supabase PostgreSQL対応

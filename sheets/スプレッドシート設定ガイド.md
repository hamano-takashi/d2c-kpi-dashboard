# Google スプレッドシート設定ガイド

## シート構成

```
シート1: 📝 入力        ← クライアントが入力するシート（シンプル）
シート2: 📊 ダッシュボード ← 自動計算で表示
シート3: 🎯 詳細データ   ← 全KPIマスター（非表示推奨）
```

## セットアップ手順

### Step 1: 新規スプレッドシート作成

1. Google スプレッドシートを新規作成
2. シート名を「入力」に変更
3. 「+」ボタンで「ダッシュボード」「詳細データ」シートを追加

### Step 2: 入力シートの設定

以下の構成で作成（黄色背景は入力セル）：

```
A列: 項目名
B列: 目標値（固定）
C列: 実績値（入力欄）← ここだけ入力
D列: 達成率（自動計算）
E列: 判定（自動計算）
```

#### 関数設定

**D列（達成率）の関数:**
```
=IF(C2="","",ROUND(C2/B2*100,1)&"%")
```

**E列（判定）の関数:**
```
=IF(C2="","",IF(C2/B2>=1,"✅達成",IF(C2/B2>=0.85,"🔵順調",IF(C2/B2>=0.7,"🟡要改善","🔴要対策"))))
```

### Step 3: ダッシュボードシートの設定

#### KGIサマリー（自動参照）

| セル | 内容 | 関数 |
|------|------|------|
| B2 | 年商実績 | `='入力'!C2` |
| C2 | 年商達成率 | `=ROUND(B2/1300000000*100,1)&"%"` |
| B3 | 営業利益実績 | `='入力'!C3` |
| C3 | 営業利益達成率 | `=ROUND(B3/195000000*100,1)&"%"` |

#### 自動計算KPI

| KPI | 関数 |
|-----|------|
| 粗利率 | `=ROUND('入力'!C4/'入力'!C2*100,1)&"%"` |
| 営業利益率 | `=ROUND('入力'!C3/'入力'!C2*100,1)&"%"` |
| ROAS | `=ROUND('入力'!C13/'入力'!C12*100,0)&"%"` |
| LTV÷CPA | `=ROUND('入力'!C17/'入力'!C18,2)&"倍"` |
| LTV-CPA | `='入力'!C17-'入力'!C18&"円"` |

### Step 4: 条件付き書式

達成率セルに色付けする条件付き書式：

1. 達成率セル範囲を選択
2. 「表示形式」→「条件付き書式」
3. 以下のルールを追加：

| 条件 | 色 |
|------|-----|
| 100以上 | 緑（#d1fae5） |
| 85以上 | 青（#dbeafe） |
| 70以上 | 黄（#fef3c7） |
| 70未満 | 赤（#fee2e2） |

### Step 5: 入力セルの保護

1. 「データ」→「シートと範囲を保護」
2. 入力シートのC列以外を保護
3. クライアントはC列のみ編集可能に

## 自動計算される指標

以下の指標は入力値から自動計算されます：

| 指標 | 計算式 |
|------|--------|
| 粗利率 | 粗利 ÷ 年商 |
| 営業利益率 | 営業利益 ÷ 年商 |
| ROAS | 広告経由売上 ÷ 総広告費 |
| LTV÷CPA | LTV ÷ CPA |
| LTV-CPA | LTV - CPA |
| 新規売上 | 年商 × (1 - リピート率) |
| リピート売上 | 年商 × リピート率 |
| チャネル構成比 | 各チャネル売上 ÷ 年商 |
| 顧客数 | 年商 ÷ LTV |

## クライアント向け説明

「黄色のセルに数字を入力するだけで、達成率やグラフが自動で更新されます」
